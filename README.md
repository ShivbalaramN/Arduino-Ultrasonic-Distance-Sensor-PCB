**Arduino Ultrasonic Distance Sensor PCB**

**📌 Project Overview**

This project presents a custom PCB design for an ultrasonic distance measurement system using an Arduino Uno and the HC-SR04.
The PCB was designed using KiCad to interface the ultrasonic sensor with the Arduino board and provide output through an LED indicator.
The system measures the distance between the sensor and an object and triggers an LED when the object is within a predefined range.

**⚙️ Features**

Ultrasonic distance measurement

LED indication for nearby objects

Custom PCB designed in KiCad

Simple and low-cost hardware implementation

Beginner-friendly embedded system project

**🧩 Components Used**

Arduino Uno
HC-SR04
LED
Resistor (220Ω)
Power input connector
PCB designed using KiCad
